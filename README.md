# 📝 Django BlogSite

A fully functional blog application built using **Django 4**, as part of the Django Masterclass Course

## 🚀 Features

- User registration, login, logout
- Create, update, delete blog posts
- Categories & tags
- Comment and system
- Search Functionality
- likes and Bookmarks
- Admin panel for managing content

## 📚 Technologies Used

- Python 3.x
- Django 4.x
- SQLite (default DB)
- HTML, CSS, Bootstrap
- Django Template Language
